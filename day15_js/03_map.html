map : key/value 형태의 데이터를 한개의 배열로 만들어서 다시 배열에 저장하는 방식

  • 선언 및 초기화 : new Map(), new Map(['key','value'],['key','value'],['key','value'])->이중배열형식
  • 2차원 배열과 비슷한 구조, 기본적으로 중첩 구조를 가지고 있기 때문에 key 중복환인으 필수!!
  • key를 식별자로 사용하느 get(keyName), set(keyName, value)메서드 제공
  • key의 타입은 문자로 하는 것이 원칙=> key의 타입이 달라지면 관리가 어려워짐
  • map.set(key, value):키에 대한 값을 변경
  • map.get(key): 키에 해당하는 value를 리턴
  • map.delete(key)/has()/forEach/entries()/size
