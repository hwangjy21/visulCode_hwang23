배열의 순환 구조를 이용하여 특정 기능을 할 수 있는 메서드 제공

  • array.reduce(callback function(total, value, index, self),[init value]) 연산의 누적값을 리턴
  • array.reduceRight(reduce와 동일) : 배열의 값을 우측에서 좌측 방향으로 대입
  • array.every(callback function(value, index, self)) 콜백함수의 리턴 조건에 모든 배열의 값을 만족하면 true을 리턴 아니면 false를 리턴
  • array.some(callback function(value, index, self)); 콜백함수의 리턴조건에 만족하는 배열값이 존재하면 true, 아니면 false를 flxjs
  • array.indexof('str',[start index]): 해당 문자열이 존재하는 배열의 index를 리턴
  • array.find(callback function(value, index, self)) 검색 조건에 만족하는 첫번쨰 배열값 리턴
  • rray.findIndex(callback function(value, index, self)) 검색 조건에 만족하는 첫번쨰 배열의 index를 리턴
  • array.from('객체') : 순서가 있거나, 순환 구조를 갖고 있는 객체를 배열로 변환
  • array.keys() : 배열이 사용중인 key(식별자 : index)를 검색하여 새로운 배열로 리턴
  • array.entries() : index:vaule의 pair 구조의 값을 만들어 새로운 배열에 담아 리턴
  • array.cludes("찾고 싶은 값"): 배열의 찾는 값이 존재하면 ture를 리턴
