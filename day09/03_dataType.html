변수나 상수에 할당할 수 있는 값의 종류

  • 숫자 : Number => 정수, 실수, NaN(not a number :숫자가 아님)
  • 문자 :String => "",'',백틱으로 감싼 값들
  • 불리언 : Boolean =true/flase
  • 객체 :Object => 1개 이상의 데이터가 특정 구조르 ㄹ가지고 있는 형태 {키 : 값} ({} : 아무 것도 없는 사실도 1개의 객체로 취급)
  • 배열 : Array =>[1,2,3,.........] 형태로 이루어진 데이터 집합 (순서, 길이가 있음 )
  • 알 수 없는 값 : null => 뭔가 있긴한데 파악이 안됨
  • 미지정된 값 : undfined => 값이 할당되지 않음
  • 데이터 타입을 확인 명령어 : typrof 변수 /값

    • 변수의 관리

    1. 기본 자료형 : 값이 메모리의 주소값이 가리크는 곳에 직접 할당 number, Boolean,string (참조형 자료이지만 사용편의상 같이 사용)
    2. 참조 자료형 (Referebce Type) : 값이 저장되어 있는 주소를 할당 String, object, Array, function, null, nudefined
    3. == 같다(값만 비교), !=같지 않다 ===같다(값과 타입이 같은지 비교) !==같다(값과 타입이 같지 않는지 비교)
